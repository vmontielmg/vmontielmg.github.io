VUE.JS

PURPOSE

According to Wikipedia, Vue.js is an open-source JavaScript framework for building user interfaces and applications. It was created by former Google employee, Evan You. While working at Google, You had worked with another popular JavaScript framework called AngularJS which is what inspired him to create Vue. He liked Angular but wanted to create a framework of his own with the features he liked about Angular but a much lighter version of it and simpler in design.

The purpose of Vue.js and other JavaScript frameworks is to improve code by making it more efficient while writing less code. Compared to other popular frameworks such as the previously mentioned AngularJS, React.js, Node.js, etc., Vue.js is similar in what it can do but what sets it apart from the others is the simplicity of it. Due to its syntax being very similar to Javascript, learning Vue is possible with the basic knowledge of plain old vanilla JavaScript. This is essential for learning a framework because the learning curve for Vue is much easier compared to Angular and other frameworks.

Some features about Vue.js that are enticing for its customers are:

  • Flexibility: Vue.js allows its users to write their code in HTML or JavaScript files. Helps other JavaScript framework users understand.
  • Simplicity: (previously mentioned) Its syntax is very similar to vanilla JavaScript which makes it easier to learn for programmers with different experience levels.
  • Small size: to download Vue.js, it takes no time to download it due to its small 18-21kb size compared to Angular.js(143kb), React.js(up to 43kb), and others. Based on a 2017 article.

LEARN

The actual Vue.js website is also very helpful. It includes many code snippets to show examples which I personally find the most helpful. The website shows the code for the HTML and JavaScript files to show how Vue is used in those files and it shows how similar the syntax is between Vue and JavaScript. This is very essential to learning a new or first-time JavaScript framework because the syntax is familiar. This website effectively shows how to create a Vue instance, which is needed to create a Vue application. There are also great examples of event handling and modifiers, components which are Vue instances, and more.

Tutorialspoint is a really helpful website that has helped me through many questions I have had in the past about anything and everything programming. I have used their tutorials for different topics like C++, Java, HTML/CSS, JavaScript, and now Vue.js.

In the Vue.js tutorial, you can learn how to download and include the script tag directly in your HTML file to include Vue into your projects. This website has about the same as the actual vuejs.org website does but what I like about tutorialspoint is that it shows programs using what you are learning. There is even a section for examples which is really useful to see how the code works. Since I have used tutorialspoint in the past, I feel more comfortable using this website for tutorials because it includes great examples and the format of the tutorials is clear.

Websites like vuejs.org and tutorialspoint.com are great examples of free resources that can help any web developer/designer learn about vue.js and more. As well as other online resources available on the internet, there are also Youtube video lectures on Vue.js and how to use it. With all the free resources we have today, learning new technologies and topics is at our fingertips and can expand our knowledge which in turn can open up possibilities for us.

COMMUNITY

Another amazing detail about Vue is the community behind it. Vue Community is accessible through their own website which has links to different ways to connect to people. They have a forum where you can ask questions and looks at questions other people have had. They also have a Discord chat so you can chat with the community at any time.

Another cool feature they have is a GitHub where you can report bugs or request a feature which is really unique and strengthens that community feel.
